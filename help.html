SteamBuddy Help & Features

Welcome to SteamBuddy! This extension automatically claims free games from Steam and Epic Games Store. Below, you'll find explanations for all features.

🎮 Steam Auto-Claim

Automatically finds and claims all free games from Steam Store based on your schedule.

  • Searches Steam Store for free games with -100% discount.
  • Skips games already in your library.
  • Supports multiple frequencies: Hourly, 6h, 12h, Daily, or On Browser Start.
  • Automatically handles age verification gates.
  • Discord notifications with game lists on success/failure.

Tip: Requires you to be logged into Steam Store.

🎮 Epic Games Auto-Claim

Automatically finds and claims all free games from Epic Games Store.

  • Detects games marked as "Free Now".
  • Navigates to each game page and clicks claim.
  • Uses same schedule as Steam auto-claim.
  • Discord notifications with game lists on success/failure.

Tip: Requires you to be logged into Epic Games Store.

🚀 Open in Steam Client (Redirect)

Automatically redirects Steam Store links to open directly in the Steam client using the steam://store/[APPID] protocol.

  • Works on game pages like https://store.steampowered.com/app/12345/.
  • Handles PatchBot redirects (e.g., affiliate links).
  • Toggle: Enable/disable via the main switch in the popup.

Tip: If Steam isn't running, it will launch the client.

🛡️ Auto Age Verification

Bypasses Steam's age gate on mature-rated games by auto-filling a default birthdate (18+ years ago) and submitting the form.

  • Triggers on pages like https://store.steampowered.com/agecheck/app/[APPID]/.
  • Sets: Day=1, Month=January, Year=Current Year - 18.
  • Always on when redirect is enabled.

Note: This is for convenience; ensure you're of legal age to view content.

⭐ Ratings Badge (Steam Reviews)

Displays a purple badge below the game title showing Steam user review summary.

  • Fetches recent reviews via Steam API.
  • Shows: Positive percentage, score description (e.g., "Very Positive"), and total review count.
  • Example: "Steam-Reviews: 85% Very Positive (12,345 Reviews)".
  • Toggle: Enable/disable in popup; reloads Steam tabs on change.

Fallback: "Not available" if data can't be loaded.

💬 Discord Webhook Notifications

Sends a message to your Discord server whenever auto-claim runs, with the list of claimed games.

  • Configure in Settings tab by entering a Discord Webhook URL.
  • Messages include game titles, color-coded by platform (Steam/Epic).
  • Sends both success messages (games claimed) and status messages (no games available).
  • Error notifications if auto-claim fails.

Tip: How to get a Discord Webhook URL: Right-click your channel → Edit Channel → Integrations → Webhooks → New Webhook

Troubleshooting

  • Auto-claim not working? Ensure you're logged into Steam and/or Epic Games Store.
  • Discord messages not sent? Check your webhook URL is correct and not expired.
  • Redirects not working? Ensure Steam client is installed and the main toggle is ON.
  • Check browser console (F12) for detailed error messages.

Having issues? Open an issue on GitHub!
